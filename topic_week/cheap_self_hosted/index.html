I like Intel NUCs if you have need for x86 hw-accelerated workloads. I run a pretty intense media system at home run on Kubernetes, mainly around the use of Plex and Ersatz, and the NUCs are key; a single i5 unit can support many simultaneous transcodes with the iGPU, and doesn't require the footprint of a PC transcoding in software (many cores at high speed required for 4k transcodes) or with a dedicated GPU doing hw-acceleration (such as leveraging an old Nvidia Geforce card, which is software limited through drivers as to how many simultaneous transcodes is allowed at once).
